Как устроен численный поиск

Для Уилсона и Agresti-Coull нельзя выразить n явно. Поэтому мы перебираем n = 1, 2, 3, ... и для каждого считаем полуширину интервала. Как только полуширина становится ≤ заданного E — останавливаемся. Это и есть нужный n.

Уилсон, полуширина:
центр = (p̂ + z²/2n) / (1 + z²/n)
EW = z · √(p̂(1−p̂)/n + z²/4n²) / (1 + z²/n)
Agresti-Coull, полуширина:
ñ = n + z², p̃ = (np̂ + z²/2) / ñ
EAC = z · √(p̃(1−p̃) / ñ)
